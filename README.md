# NEPIRE-pwn
Basic CTF-PWN Tool collection

### 必要环境
```
docker,python
```

#### INSTALL
```
git clone https://github.com/Nepire/nepire-pwn.git ~/nepire-pwn/
cd ~/nepire-pwn
./install
```

#### RUN
```
$ nepire-pwn
```

# NEPIRE-pwn
Basic CTF-PWN Tool collection

### 必要环境
```
docker,python
```

#### INSTALL
```
wget -q -O- https://raw.githubusercontent.com/Nepire/nepire-pwn/master/install | sh
```

#### RUN
```
$ nepire-pwn
```

#### up_process
每次进入nepire-pwn时会从本机的nepire-pwn/process文件夹里的文件同步至nepire-pwn
```
$ cp filename ~/nepire-pwn/process/
```

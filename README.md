# nepire-pwn
nepire的pwn解题环境

### 必要环境
```
docker
```
### 前置环境安装(ubuntu)
```
sudo apt-get install docker.io
```
#### 测试必要环境安装结果
```
docker -v
```

#### INSTALL
```
git clone https://github.com/Nepire/nepire-pwn.git ~/
cd ~/nepire-pwn
./install
```

#### RUN
```
$ nepire-pwn
```

# NEPIRE-pwn
Basic CTF-PWN Tool collection

### 必要环境
```
docker,python
```

#### INSTALL
```
git clone https://github.com/Nepire/nepire-pwn.git ~/nepire-pwn/
cd ~/nepire-pwn
chmod a+x install
./install
```

#### RUN
```
$ nepire-pwn
```

#### up_process(每次进入nepire-pwn时会将~/nepire-pwn/process/里的文件上传至环境中)
```
$ cp filename ~/nepire-pwn/process/
```
